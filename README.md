# DeepFake-Image-Detection

This project implements a privacy-aware DeepFake image detection framework using:

- Local Binary Patterns (LBP)
- Support Vector Machine (SVM)
- CNN
- MobileNetV2 Transfer Learning
- Ensemble Learning

## Dataset
CASIA v2.0

## Results
- LBP + SVM: 62.33%
- CNN: 49.25%
- MobileNetV2: 81.58%
